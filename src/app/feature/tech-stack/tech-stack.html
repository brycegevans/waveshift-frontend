AI-Powered Sports Data Assistant

A natural language interface to sports data built with Python and OpenAI — ask it anything from player stats to game predictions.

  • Backend in Python with Flask + FastAPI, using OpenAI GPT APIs
  • Ingests and indexes large datasets of real-time and historical sports data
  • Supports natural queries like “Who had the most rebounds in the 2023 playoffs?”
